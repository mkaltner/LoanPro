# Changelog

## 1.0.0

- Rebuilt LoanPro as a modern single-module Android project.
- Restored loan calculator behavior for price, payment, term, and interest calculations.
- Improved numeric entry, decimal, percent, backspace, and clear behavior.
- Added clearer invalid calculation messages for common impossible scenarios.
- Added adaptive launcher icons, themed icon support, and Play/F-Droid listing icon.
- Removed legacy paid/lite wrappers, licensing checks, analytics, and unused permissions.
