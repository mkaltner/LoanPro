# Changelog

## 1.0.2

- Removed Android Gradle Plugin dependency metadata from release APKs for F-Droid reproducible build compatibility.

## 1.0.0

- Rebuilt LoanPro as a modern open-source Android app.
- Restored fast amount, payment, term, and interest calculations.
- Improved number entry, percent, decimal, backspace, clear, and repeated-use behavior.
- Added useful invalid-result messages for common impossible scenarios.
- Added adaptive launcher icons, themed icon support, and refreshed listing artwork.
- Removed analytics, unused permissions, licensing checks, and legacy wrapper projects.
- Added release metadata, privacy notes, changelog, and Apache-2.0 license.
